    **7-Step 구현 (L2CrossTrade 쌍) — 각 step에 L2 검증 포함:**

    **Step 1: Deploy L2CrossTrade impl (creation Deposit Tx)**
    - `implAddr := crypto.CreateAddress(deployerAddr, l2Nonce)` — 주소 예측
    - `sendDepositCreation(ctx, portal, opts, l1Client, implBytecode, 3_000_000, logger)` — L1 tx
    - `waitForContractCode(ctx, l2Client, implAddr, logger)` — **L2 creation 검증 (getCode polling, per D-04)**
    - logger.Infof("Step 1: L2CrossTrade impl deployed at %s", implAddr.Hex())

    **Step 2: Deploy L2CrossTradeProxy (creation Deposit Tx)**
    - `proxyAddr := crypto.CreateAddress(deployerAddr, l2Nonce+1)` — 주소 예측
    - `sendDepositCreation(ctx, portal, opts, l1Client, proxyBytecode, 3_000_000, logger)` — L1 tx
    - `waitForContractCode(ctx, l2Client, proxyAddr, logger)` — **L2 creation 검증 (getCode polling, per D-04)**
    - logger.Infof("Step 2: L2CrossTradeProxy deployed at %s", proxyAddr.Hex())

    **Step 3: setAliveImplementation2(implAddr, true) [PRD 누락 보완, Pitfall 2 방지]**

    - logger.Infof("Step 3: setAliveImplementation2 for impl %s — L2 verified", implAddr.Hex())

      ```go
      implABI, _ := abi.JSON(strings.NewReader(implABIJSON))
      var selectors [][4]byte
      for _, method := range implABI.Methods {
          var sel [4]byte
          copy(sel[:], method.ID[:4])
          selectors = append(selectors, sel)
      }
      ```
    - `calldata, _ := proxyABI.Pack("setSelectorImplementations2", selectors, implAddr)`

      ```go
      if len(selectors) > 0 {
      ```
