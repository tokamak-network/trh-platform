- DeFi 프리셋 선택 -> Step 2 (Local Docker, chain name, L1 RPC, seed phrase) 입력 -> Step 3 Review -> Deploy 클릭

주의사항:
- Electron 앱의 WebContentsView는 `app.windows()` 의 두 번째 window로 나타남 (첫 번째는 main BrowserWindow)
- 만약 WebContentsView가 별도 window로 안 나타나면, main window 내의 iframe/webview로 접근해야 할 수 있음 — `page.frame()` 사용
- Backend API 호출은 직접 `fetch()` 사용 (Playwright page가 아닌 Node.js fetch)
- `loginBackend()` 함수 재사용하여 JWT 획득 후 Authorization 헤더에 포함
- deploy-helper.ts와 stack-resolver.ts의 기존 함수를 import하여 재사용
