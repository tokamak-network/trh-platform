# EFP-02 FailedToDeploy — Root Cause & Fix

**날짜**: 2026-05-15  
**영향 범위**: EFP-02 실패 → serial 모드로 EFP-03~11 전체 스킵

## 증상

두 차례 연속 실패 (2026-05-14 00:54 KST, 2026-05-15 09:53 KST):
- EFP-02 `should deploy L2 chain to AWS EKS successfully` → `FailedToDeploy`
- Stage B Terraform apply 실패

## 로그 증거

```
[2026-05-15T01:11:54] module.k8s.terraform_data.thanos_stack_values (local-exec):
Error: Required environment variable 'stack_op_geth_image_tag' is undefined
```

실제 `.envrc` 파일 (`trh_backend_storage` 볼륨에서 확인):
```
export TF_VAR_stack_op_geth_image_tag=""
export TF_VAR_stack_thanos_stack_image_tag=""
```

실제 `settings.json`:
```json
{
  "network": "",
  ...
}
```

## 근본 원인

**trh-sdk `pkg/stacks/thanos/deploy_chain.go`의 Stage B `makeTerraformEnvFile` 호출 버그**

```go
// Stage A (line 281-282) — 정상: t.network 사용
ThanosStackImageTag: constants.DockerImageTag[t.network].ThanosStackImageTag,
OpGethImageTag:      constants.DockerImageTag[t.network].OpGethImageTag,

// Stage B (line 465-466) — 버그: t.deployConfig.Network 사용
ThanosStackImageTag: constants.DockerImageTag[t.deployConfig.Network].ThanosStackImageTag,
OpGethImageTag:      constants.DockerImageTag[t.deployConfig.Network].OpGethImageTag,
```

인과 체인:
1. `DeployAWSStageB`가 `t.reloadDeployConfig()` 호출 → `settings.json` 읽기
2. `settings.json`의 `"network": ""` (network 필드는 디스크에 저장되지 않음)
3. `constants.DockerImageTag[""]` → Go map zero-value → `OpGethImageTag = ""`
4. `.envrc`에 `export TF_VAR_stack_op_geth_image_tag=""` 기록
5. `terraform apply` 실행 → provisioner 환경변수로 빈 문자열 전달
6. `generate-thanos-stack-values.sh`의 `reqenv "stack_op_geth_image_tag"` → `[ -z "" ]` = true → exit 1
7. Terraform apply 실패 → Stage B 실패 → `FailedToDeploy`

`t.network`는 `NewThanosStack` 생성자 파라미터로 설정되어 항상 올바른 값(`"testnet"` 또는 `"mainnet"`)을 가짐.

## 수정

**trh-sdk `pkg/stacks/thanos/deploy_chain.go` 465-466번째 줄**:

```go
// Before (buggy):
ThanosStackImageTag: constants.DockerImageTag[t.deployConfig.Network].ThanosStackImageTag,
OpGethImageTag:      constants.DockerImageTag[t.deployConfig.Network].OpGethImageTag,

// After (fixed):
ThanosStackImageTag: constants.DockerImageTag[t.network].ThanosStackImageTag,
OpGethImageTag:      constants.DockerImageTag[t.network].OpGethImageTag,
```

버그 도입 커밋: `2df2e5b` — Stage A/B 분리 시 Stage B의 `makeTerraformEnvFile` 호출에 잘못된 필드 사용
