Deployment Sequence — Backend → SDK Detail
Platform UI POST /api/v1/stacks/thanos/preset-deploy
{ presetId, chainName, network, feeToken, seedPhrase, l1RpcUrl, beaconUrl, awsCredentials, overrides }
Backend BIP44 Key Derivation (seed phrase → 5 accounts)
admin · proposer · batcher · challenger · sequencer — paths m/44'/60'/0'/0/{0-4}
Backend L1 Funding Status Check
Poll sequencer / batcher / proposer balances — testnet: 0.5 ETH · mainnet: 2 ETH minimum
Backend trh-sdk Invocation
thanos deploy --preset <id> --fee-token <token> --network <net> --chain-name <name> ...
trh-sdk Deploy Config Initialization (preset-specific overrides)
Apply preset defaults for batchSubmissionFrequency · outputRootFrequency · l2BlockTime
trh-sdk L1 Contract Deployment
OptimismPortal · L1CrossDomainMessenger · L1StandardBridge · L2OutputOracle · SystemConfig
trh-sdk L2 Genesis + Chain Bootstrap
Insert GenesisPredeploys (preset-specific) → Start Sequencer · Batcher · Proposer → Helm deploy
trh-sdk Preset Module Auto-Install
Based on PresetModules map — bridge · blockExplorer · monitoring · uptimeService (uptimeService installs automatically; others configurable in the Integration tab post-deploy)
Backend Save Deployment Result + Status Response
Returns deploymentId, stackId, rollupRpcUrl, explorerUrl, bridgeUrl
