딜레마 구조 복기

TON 수수료의 문제

L2가 TON을 수수료로 요구하면:
① 사용자가 TON을 보유해야 함
② TON 가격 변동 → 예측 불가한 수수료
③ 생태계 초기 단계에서 수용성 낮음

시뇨리지 보조로 해결

시퀀서가 시뇨리지 수입으로 실질 운영 비용 충당
→ TON 수수료를 낮게 유지해도 운영 가능
→ 사용자 부담 감소, 채택 장벽 완화

기본 검증 모델

12 / 17
기대 보수 비교
검증 시 기대 보수
$X \cdot C + VR - VC$
비검증 시 기대 보수
$-X \cdot L + (1-X) \cdot VR$

딜레마 임계값 도출

검증 ≥ 비검증 조건에서 X를 구하면:

$X \cdot C + VR - VC \geq -X \cdot L + (1-X) \cdot VR$

정리 → $X \leq \dfrac{VC - VR}{C + L + VR}$

수치 예시
파라미터값
C (적발 보상)10 TON
L (시퀀서 손실)10 TON
VC (검증 비용)20 TON
VR (비검증 보상)10 TON
딜레마 임계값: $X \leq \dfrac{1}{2}$
시퀀서가 50% 이상 부정직해야 검증이 합리적
한계: $L = $ 외부 파라미터로 조정 어려움. 실질적 보안 개선 수단이 없음.

스테이킹 검증 모델

13 / 17
기대 보수 비교
검증 시 기대 보수 (기본과 동일)
$X \cdot C + VR - VC$
비검증 시 기대 보수 (변경)
$-X \cdot A \cdot S + (1-X) \cdot VR$

핵심 변화

$L = A \cdot S$ (슬래싱이 손실 결정)
A = 슬래싱 비율 (거버넌스로 조정 가능)
S = 스테이킹 TON 수량

기본 모델 대비 개선점

수치 예시

S = 1,000 TON, A = 10% → L = 100 TON
A = 50% → L = 500 TON (더 높은 억제력)

개선: A 파라미터 조정 가능

거버넌스로 슬래싱 비율 A를 조정해
손실 L을 원하는 수준으로 제어 가능

딜레마 임계값: $X \leq \dfrac{1}{2}$ (기본 모델과 동일)
임계값 자체는 변하지 않지만 A 조정으로 유연성 확보
한계: 임계 X = 1/2로 기본 모델과 동일. 더 낮은 임계값을 위해 추가 메커니즘 필요.

스테이킹 + 빠른 출금 검증 모델

14 / 17
기대 보수 비교
검증 시 기대 보수 (동일)
$X \cdot C + VR - VC$
비검증 시 기대 보수 (FW 추가)
$-X \cdot (A \cdot S + FW) + (1-X) \cdot VR$

FW의 역할

빠른 출금에 제공한 TON = FW
부정직 시퀀서 적발 시 FW도 위험에 처함
→ 손실 = $A \cdot S + FW$ (더 큼)

임계값 계산 (수치 예시)
파라미터값
C (적발 보상)10
A·S (슬래싱 손실)10
FW (빠른 출금 TON)100
VC (검증 비용)20
VR (비검증 보상)10
딜레마 임계값: $X \leq \dfrac{2}{13} \approx 0.154$

기본/스테이킹 모델(0.5) 대비 3.25배 더 낮음
FW가 클수록 임계값은 더 낮아짐 → 더 안전
