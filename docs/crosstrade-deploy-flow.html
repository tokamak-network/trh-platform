CrossTrade 배포 플로우

DeFi Preset을 선택하면 무슨 일이 일어나는가

🖥️
Step 1
DeFi Preset 선택
TRH Platform 앱에서 L2 체인 유형을 고릅니다.
DeFi 또는 Full을 선택하면 CrossTrade가 자동으로 포함됩니다.
trh-platform (Electron)
⛓️
Step 2
새로운 L2 블록체인 배포
OP Stack 기반의 새 L2 네트워크가 생성됩니다.
배포가 완료되면 CrossTrade 통합이 자동으로 시작됩니다.
trh-backend + trh-sdk
📦
Step 3
CrossTrade 스마트 컨트랙트를 L2에 배포
새 L2에서 직접 배포하는 것이 아니라, L1(Sepolia)의 브릿지를 통해 배포합니다.
이 방식으로 L2의 보안과 브릿지 연결이 보장됩니다.
1
L2CrossTrade 배포 — L2↔L1 토큰 교환용 컨트랙트
2
L2toL2CrossTrade 배포 — L2↔L2 토큰 교환용 컨트랙트
3
각 컨트랙트에 USDC, ETH 토큰 쌍 등록
각 배포 후 L2에서 실제로 실행됐는지 자동으로 검증합니다.
trh-sdk — OptimismPortal Deposit Tx
🔗
Step 4
L1에 새 L2 체인 정보 등록
Sepolia에 이미 배포된 CrossTrade 인프라 컨트랙트에
새로 만든 L2의 주소를 알려줍니다.
📝 L1CrossTradeProxy 등록
📝 L2toL2CrossTradeL1 등록
이 단계가 완료되어야 L1과 L2 간 토큰 이동이 가능합니다.
trh-backend — L1 setChainInfo
⚙️
Step 5
CrossTrade dApp 환경 자동 설정 및 실행
배포된 컨트랙트 주소와 RPC URL을 기반으로
dApp 설정 파일이 자동으로 생성되고 컨테이너가 실행됩니다.
📄 .env.crosstrade 자동 생성
🐳 Docker 컨테이너 실행
trh-backend + docker-compose.crosstrade.yml
🔍
Step 6
Platform UI에 dApp URL 표시
Integrations 탭의 CrossTrade 카드에 dApp 주소가 나타납니다.
버튼 하나로 dApp을 바로 열거나 주소를 복사할 수 있습니다.
trh-platform-ui — CrossTradeCard
🎉

CrossTrade 배포 완료

DeFi Preset 선택 하나로 L2 컨트랙트 배포, L1 등록,
dApp 실행까지 모두 자동으로 처리됩니다.
운영자가 추가로 할 일은 없습니다.
